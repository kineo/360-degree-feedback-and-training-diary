Personal Development Plan Additional Fields

The field names that appear below a student's Personal Development Plan, and allow each student to store additional information about their plan. To create new fields enter text in the blank fields. If you need to create more fields click the "Add 2 more additional fields" button to add more blank fields to the form.

Checking the "delete this field" checkbox will remove fields. Note that this will also permenantly delete any field values that students may have already entered. You can add new field names and modify existing ones without affecting existing student entries.
