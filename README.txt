Description
------------

These modules allows organisations to run 360 appraisals and training diaries directly within Moodle. The functionality of the modules includes:

-	custom definition of competences and ratings scale 
-	a range of reviewers to be included such as managers, peers, staff and customers
-	automatic emailing and online completion of review
-	full reporting on desired ratings by competency and actual ratings
-	anonymous reporting
-	filtering by reviewer type
-	spider diagram summaries highlighting strengths and weaknesses
-	training diaries based on a selected number of carried over competencies

Requirements
-------------

* Moodle 1.9

Installation
-------------

1- Unpack the module into your moodle install in order to create a
   mod/threesixty directory
   mod/trdiary directory

2- Visit the /admin/index.php page to trigger the database installation.

3- (Optional) Change the default options in the activity modules
   configuration.
