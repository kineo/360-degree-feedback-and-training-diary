Reflective Log Field Names

The field names that appear when a student creates a reflective log entry. To create new fields enter text in the blank fields. If you need to create more fields click the "Add 2 more reflective log fields" button to add more blank fields.

Checking the "delete this field" checkbox will remove fields. Note that this will also permenantly delete any field values that students may have already entered. You can add new field names and modify existing ones without affecting existing reflective log entries.
